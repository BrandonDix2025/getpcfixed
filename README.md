# GetPCFixed

Your PC deserves better than error messages and slow boots.

GetPCFixed detects, diagnoses, and fixes the most common Windows
problems everyday people face — always asking before changing anything.

Built for the 1 billion Windows users the enterprise tools forgot.

---

## What It Does

- Scans your PC health — CPU, RAM, disk, system info
- AI diagnosis — plain English explanation of what's wrong
- Cleans junk files — frees up wasted space
- Fixes startup programs — speeds up boot time
- Remembers everything — full history log of every scan and fix

---

## How To Run It

1. Make sure Python is installed
2. Install the required packages:
pip install psutil anthropic pyqt5 python-dotenv
3. Create a .env file in the project folder:
ANTHROPIC_API_KEY=your-key-here
4. Run the app:
python app.py
